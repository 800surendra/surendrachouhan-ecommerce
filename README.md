## Documentation

### Hindi

यह परियोजना "सुरेंद्र चौहान ई-कॉमर्स स्टोर" है। यह एक ऑनलाइन स्टोर है जहाँ उपयोगकर्ता विभिन्न उत्पादों की खोज और खरीद कर सकते हैं। यहाँ पर परियोजना की संरचना और उपयोग के निर्देश दिए गए हैं।  

### सेटअप निर्देश:
1. **कोड को क्लोन करें:**  
   ```bash
   git clone https://github.com/800surendra/surendrachouhan-ecommerce.git
   cd surendrachouhan-ecommerce
   ```  
2. **आवश्यक पैकेज स्थापित करें:**  
   ```bash
   npm install
   ```  
3. **ऐप्लिकेशन चलाएँ:**  
   ```bash
   npm start
   ```

### अंग्रेजी

This project is the "Surendrachouhan E-commerce Store". It is an online store where users can browse and purchase various products. Below are the instructions for setting up and running the project.

### Setup Instructions:
1. **Clone the code:**  
   ```bash
   git clone https://github.com/800surendra/surendrachouhan-ecommerce.git
   cd surendrachouhan-ecommerce
   ```  
2. **Install required packages:**  
   ```bash
   npm install
   ```  
3. **Run the application:**  
   ```bash
   npm start
   ```  

### Features:
- User registration and login  
- Product browsing  
- Add products to cart  
- Order management
